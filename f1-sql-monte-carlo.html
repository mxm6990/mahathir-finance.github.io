Model Disclaimer

This model provides probabilistic estimates of Formula 1 race outcomes and championship standings using historical performance metrics derived from structured race data.

The simulation is based on simplified assumptions and does not account for several real-world factors that can significantly impact race outcomes.

⚠️ Key Limitations

1. No Race Incidents Modeled

  • Safety cars
  • Red flags
  • Collisions
  • Mechanical failures (DNFs)

👉 These events introduce randomness not captured in the model.

2. No Strategy Modeling

  • Pit stop timing
  • Tire strategy
  • Undercuts / overcuts

👉 Strategy is a major determinant of race results but is excluded.

3. No Dynamic Weather Simulation

  • Weather is treated as static per race
  • No intra-race changes (e.g., dry → wet transitions)

4. Limited Data Scope (Current Version)

  • Based on a small sample dataset
  • Results improve with more historical race data

5. Simplified Driver Performance Modeling

Uses aggregate metrics such as:

  • average finish
  • racecraft (positions gained)
  • composure (early vs late lap pace)

👉 Does not model teammate interactions, car upgrades, or track-specific strengths.

🧠 Interpretation Guidance

Results should be interpreted as directional probabilities, not exact predictions.

The model is best used to explore how driver performance metrics translate into probabilistic outcomes under controlled assumptions.
