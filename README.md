# 📊 Mahathir Finance Platform

A full-stack financial analysis platform combining Python-based financial modeling with an interactive web interface.

This project transforms complex financial concepts like valuation, forecasting, and loan analysis into intuitive, user-friendly tools.

🌐 Live Demo: https://mxm6990.github.io/mahathir-finance.github.io/

---

## 🚀 Features

### 📈 Valuation Engine (DCF + APV + Comps)
- Discounted Cash Flow (DCF) modeling
- Adjusted Present Value (APV) analysis
- Comparable company analysis (Comps)
- WACC calculation using CAPM inputs
- Football field valuation visualization

### 📊 Forecasting System
- Revenue and Free Cash Flow projections
- Scenario-based forecasting
- Modular financial model structure

### 🏦 Amortization Analyzer
- Multi-scenario loan comparison
- Supports FHA, Jumbo, Balloon, PMI
- Payment breakdown visualization
- Interactive charts (principal vs interest)

### 📉 Market Data Integration
- Stock price tracking
- Beta calculations (including peer beta)
- Historical price analysis
